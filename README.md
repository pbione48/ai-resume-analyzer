# AI Resume Analyzer

A Streamlit web application that analyzes resumes using OpenAI and provides structured feedback for improvement.

## Features
- Resume analysis (strengths, weaknesses, improvements)
- Bullet point rewriting
- Overall scoring system

## Tech Stack
- Python
- Streamlit
- OpenAI API

## How to Run
1. Clone repo
2. Create `.env` file:
   OPENAI_API_KEY=your_key_here
3. Install dependencies:
   pip install -r requirements.txt
4. Run:
   streamlit run app.py
